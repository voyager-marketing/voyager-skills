# Client Onboarding Checklist

## Prerequisites
- [ ] WordPress installed and accessible
- [ ] Required plugins: Voyager Orbit, Voyager Blocks, Voyager Core
- [ ] Voyager Block Theme activated

## Site Setup
- [ ] Header pattern assigned (header-default or header-mega-menu)
- [ ] Footer pattern assigned (footer-4col or footer-mega)
- [ ] Homepage built (hero + services-grid + testimonials + cta-bottom)
- [ ] Services page (services-hero + services-detail)
- [ ] About page (about-mission + team-grid + stats)
- [ ] Contact page (contact-centered)
- [ ] Blog page (blog-grid-3)

## Content
- [ ] Logo uploaded and set
- [ ] Brand colors configured in theme.json or Customizer
- [ ] Client content replaces placeholder text
- [ ] Images uploaded and optimized

## SEO
- [ ] RankMath installed and configured
- [ ] Sitemap generated
- [ ] Meta descriptions on key pages

## pSEO (if applicable)
- [ ] Service areas generated via WP-CLI
- [ ] Service area patterns assigned
- [ ] AI enrichment run

## Quality
- [ ] php health-check.php passes
- [ ] All pages reviewed in editor (no recovery warnings)
- [ ] Mobile responsive check
- [ ] Client walkthrough completed
